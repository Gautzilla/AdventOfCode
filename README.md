# AdventOfCode
Puzzle solvings for [Advent of Code](https://adventofcode.com/2021/about).

Each year is included in the solution (WIP, I started in 2021 and didn't finish any year yet).

Each year has its own `Program.cs`, where puzzles can be solved using the `DayX.Solve(part)` method, where X is the n° of the day and the `part` parameter represents the part 1 or part 2 of the puzzle.
